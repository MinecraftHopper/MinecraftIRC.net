--- layout: article title: "Force Computer To Use Nvidia Hardware" last_updated: 2018-09-03 13:42:00 +0000 ---

In some cases, if you are using a laptop, you may need to force Minecraft to use your Nvidia Graphics hardware instead of the Intel hardware.

Windows

  1. Click the Show Hiddent Icons button Looks like ^ on your taskbar, then right click the Nvidia icon and select Nvidia Control Panel
  2. In the Nvidia Control Panel, click the Manage 3D Settings option from the menu at the left, then click on the Program Settings tab.
  3. Because Minecraft runs using Java, you need to add the version of Java that Minecraft uses to this list, not the Minecraft launcher itself Click on the Add button, and in the window that opens, click the browse button.
  4. By default Minecraft uses its bundled version of Java, which can be found at C:\Program Files (x86)\Minecraft\runtime\jre-x64\1.8.0_51\bin Enter that in the address bar at the top of the Browse window select the javaw program, then hit open. Note: If you have set the game to use a different version of Java, or the bundled Java gets updated, you will need to adjust the folder you use for this step
  5. It should then autoselect what you just added, but make sure the Java you just added is selected below the "Select a program to customize option" In the list below "Specify the settings for this program" scroll down until you see the option "OpenGL rendering GPU", click the drop down menu next to it, and select your Nvidia hardware.
  6. Click on the Apply button, then close the Nvidia Control Panel and try to run the game again, you can press F3 ingame to confirm your Nvidia hardware is being used.
