Name of error java.net.SocketException
Solution True
Mojira issue NONE
Applies to Network Connectivity, Minecraft Launcher, MultiPlayer Connections
Environment Windows
Comments This issue will halt all connections the game attempts to make with the Internet.
Error log 
java.net.SocketException: Unrecognized Windows Sockets error: 87: create
Solution

Disable any running antiviruses, or restrictions in place for the host that deny connections to external sites.

You can solve this error by resetting your WinSock by using the following command in an administrator Command Prompt:Â  netsh winsock reset

At some occasions, this may fail to work, and workarounds include to reinstall .NET or use the repair tool provided by Microsoft.

Cause

The cause of this error is due to an Windows Socket error: WSA_INVALID_PARAMETER 87. See here for a more technical explanation of the issue..
