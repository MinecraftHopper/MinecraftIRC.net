Changing Java Version

Something you need (or want) to update the Java version that is bundled with the game to a newer version, let's learn how to do that.

Installing Java

Please follow the steps in this guide for instructions on how to install Java.

Updating Launcher Path

Start the Minecraft launcher if you have not already. At the top right of the Minecraft launcher should be hamburger menu, click on the icon.

Select "Launch Options" on the menu at the top.
