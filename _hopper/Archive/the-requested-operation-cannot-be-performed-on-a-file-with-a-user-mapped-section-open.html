The requested operation cannot be performed on a file with a user-mapped section open
This error can be caused by a number of issues, including client modifications, file permission errors, or an antivirus software.
How to fix this

Since this error can be caused by many things, please make sure that:

  • You are not using client modifications
  • The game has the ability to write to where it stores its files
  • You have tried disabling your antivirus software
