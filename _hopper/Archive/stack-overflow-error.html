Stack overflow error


Fatal errors such as StackOverflow, generally signify something that has gone wrong which has caused other things to come down along with it, in a constant loop until the process is terminated.

As such error can be vague, there is no absolute solution to these errors.

Exception while updating neighbours

StackOverflow Errors which contain updating neighbors at fault, are usually due to a damaged, and possibly corrupt world.

If this is the first time you have received this error, try to restart the client and reproduce the issue.

Otherwise the issue persists, and you have encountered this issue while updating a block, there may have been an issue with certain blocks in the area, which caused numerous other blocks to be affected as well.

Here are a couple of possible troubleshooting steps you can take:

  • Try and reproduce the error on a freshly generated world
  • Try to reinstall the client and or server
  • Try to recover corrupted save data
