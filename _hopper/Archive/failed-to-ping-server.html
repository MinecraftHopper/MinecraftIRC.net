Resetting your server list to default

If you are getting this error after adding a server to the game, then the game may not recognize the server, and cause it to crash.

To fix this, you may need to reset your settings to default, by following these instructions:

  1. Locate the Minecraft data folder
  2. Look for the file "servers.dat"
  3. Delete the file, which usually can be done by selecting the file and pressing 'Delete' on your keyboard.
