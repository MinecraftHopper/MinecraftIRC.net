Modded Minecraft

This page is for dealing with crashes due to running a version of Minecraft that was not produced by Mojang. These versions are often called "mods" (short for modification) and are not under Mojang's control, but we can help you fix crashes.

How to fix the problem

  • Option 1: Go clean

    • Remove mods and use a Mojang-supplied version of Minecraft (YOU WILL LOSE ANYTHING THAT IS PART OF THE MODS AND WILL NOT BE ABLE TO CONNECT TO SERVERS WITH THE MOD).
    1. Open the Launcher.
    2. Click "New Profile" in the upper right-hand corner.
    3. Change all the settings you want, but leave "Use Version" set to "Use Latest Version".
    4. Click "Save Profile".
    5. Run the launcher with the newly created profile selected in the upper right-hand corner.

  • Option 2: Revert

    • Use an earlier version of the mod that worked (YOU WILL LOSE ANYTHING ADDED IN A LATER VERSION OF THE MOD AND WILL NOT BE ABLE TO CONNECT TO SERVERS USING NEWER VERSIONS OF THE MOD).
    1. Open your .minecraft folder.
    2. Open the folder "Versions".
    3. Open the folder of the modded version you are using.
    4. Replace the [VERSIONNAMEHERE].jar with an older .jar file that worked.

  • Option 3: Update

    • Update the mod to a newer version that may have fixed the bug (YOU WILL NOT BE ABLE TO CONNECT TO SERVERS USING OLDER VERSIONS OF THE MOD).
    1. Open your .minecraft folder.
    2. Open the folder "Versions".
    3. Open the folder of the modded version you are using.
    4. Replace the [VERSIONNAMEHERE].jar with an newer .jar file of the mod.

  • Option 4: Ask for help

    • Contact the creators of the mod or others who use the mod and ask them for help.
